# Changelog

All notable changes to this project will be documented in this file.

## [1.4.0] - 2024-03-01

### Added
- Complete Flutter app structure with Material Design 3
- Login screen with phone verification
- Home screen with device list
- Device control screen with all modules
- API service with Dio
- BLE service with Flutter Blue Plus
- Offline service with Hive local storage
- OTA service for firmware updates
- Push service with JPush
- Voice service for speech recognition
- Complete technical documentation

### Features
- Bluetooth device scanning and connection
- Music control (play/pause/stop, volume, mode)
- Light control (on/off, brightness, color temperature)
- Massage control (intensity, mode, position)
- Scent control (on/off, concentration, scent type)
- Sleep data tracking and reporting
- Real-time vital signs monitoring

## [1.3.0] - 2024-02-15

### Added
- OTA firmware upgrade support
- Offline mode with local caching
- Sleep data synchronization

## [1.2.0] - 2024-01-20
