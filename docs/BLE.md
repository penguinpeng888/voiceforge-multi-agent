## 1. 概述

本文档定义睡眠舱设备与移动应用之间的蓝牙低功耗（BLE）通信协议。

### 1.1 通信模式

- **命令-响应模式**: 应用发送命令，设备返回响应
- **主动上报模式**: 设备主动上报状态变化

### 1.2 连接流程

```
扫描设备 → 连接设备 → 发现服务 → 查找特征 → 开始通信
```

## 3. 通信帧格式

### 3.1 命令帧（App → Device）

```
| 帧头 | 长度 | 命令码 | 数据载荷 | CRC8 |
|------|------|--------|----------|------|
| 2字节 | 1字节 | 1字节  | N字节    | 1字节|
```

| 字段 | 字节 | 说明 |
|------|------|------|
| 帧头 | 2 | 固定 `0xAA 0x55` |
| 长度 | 1 | 命令码 + 数据载荷的字节数 |
| 命令码 | 1 | 见命令集 |
| 数据载荷 | N | 命令参数 |
| CRC8 | 1 | CRC 校验（多项式 0x07） |

### 3.2 响应帧（Device → App）

```
| 帧头 | 长度 | 响应码 | 状态 | 数据载荷 | CRC8 |
|------|------|--------|------|----------|------|
| 2字节 | 1字节 | 1字节  | 1字节| N字节    | 1字节|
```

| 字段 | 字节 | 说明 |
|------|------|------|
| 帧头 | 2 | 固定 `0x55 0xAA` |
| 长度 | 1 | 响应码 + 状态 + 数据载荷的字节数 |
| 响应码 | 1 | 命令码 + 0x80 (如 0x01 → 0x81) |
| 状态 | 1 | 执行结果 |
| 数据载荷 | N | 响应数据 |
| CRC8 | 1 | CRC 校验 |

### 3.3 CRC8 算法

```c
uint8_t calculate_crc8(uint8_t *data, uint8_t len) {
    uint8_t crc = 0x00;
    for (int i = 0; i < len; i++) {
        crc ^= data[i];
        for (int j = 0; j < 8; j++) {
            if (crc & 0x80) {
                crc = (crc << 1) ^ 0x07;
            } else {
                crc <<= 1;
            }
        }
    }
    return crc;
}
```

## 4. 命令集

### 4.1 系统命令 (0x00-0x0F)

#### 4.1.1 心跳检测 (0x01)

**命令帧:**

```
AA 55 01 CRC
```

**响应帧:**

```
55 AA 81 01 [状态] CRC
```

| 状态 | 说明 |
|------|------|
| 0x01 | 正常 |
| 0x02 | 异常 |

---

#### 4.1.2 电源控制 (0x02)

**命令帧:**

```
AA 55 02 [模式] CRC
```

| 参数 | 字节 | 说明 |
|------|------|------|
| 模式 | 1 | 0=关机, 1=正常, 2=休眠, 3=深度休眠 |

**响应帧:**

```
55 AA 82 [状态] CRC
```

| 状态 | 说明 |
|------|------|
| 0x01 | 成功 |
| 0x02 | 设备忙碌 |
| 0x03 | 参数错误 |

---

### 4.2 音乐控制 (0x10)

#### 命令: `0x10`

**命令帧:**

```
AA 55 10 [动作] [音量] [模式] [保留] [保留] [保留] [保留] CRC
```

| 字节 | 参数 | 说明 |
|------|------|------|
| 0 | 动作 | 0=停止, 1=播放, 2=暂停 |
| 1 | 音量 | 0-100 |
| 2 | 模式 | 0=single, 1=loop, 2=random |
| 3-6 | 保留 | 音轨 ID |
| 7-8 | 保留 | 时长（分钟）|

**响应帧:**

```
55 AA 90 [状态] CRC
```

| 状态 | 说明 |
|------|------|
| 0x01 | 播放成功 |
| 0x02 | 播放失败 |
| 0x03 | 无此曲目 |

---

### 4.3 灯光控制 (0x11)

#### 命令: `0x11`

**命令帧:**

```
AA 55 11 [开关] [亮度] [R] [G] [B] [色温高] [色温低] [模式] CRC
```

| 字节 | 参数 | 说明 |
|------|------|------|
| 0 | 开关 | 0=关, 1=开 |
| 1 | 亮度 | 0-100 |
| 2-4 | RGB | 颜色值 |
| 5 | 色温高 | 色温 >> 8 |
| 6 | 色温低 | 色温 & 0xFF |
| 7 | 模式 | 0=手动, 1=渐变, 2=呼吸, 3=睡眠 |

**色温范围:** 2700K - 6500K

**响应帧:**

```
55 AA 91 [状态] CRC
```

| 状态 | 说明 |
|------|------|
| 0x01 | 灯光已开启 |
| 0x02 | 灯光已关闭 |

---

### 4.4 按摩控制 (0x12)

**命令帧:**

| 字节 | 参数 | 说明 |

| 3 | 位置 | 位图: bit0=头, bit1=颈, bit2=肩, bit3=背, bit4=腰, bit5=腿 |

**位置位图示例:** `0x3F` = 全部位置

**响应帧:**

| 0x01 | 按摩已启动 |

## 6. 数据示例

### 6.1 开启灯光

**发送:**
```
AA 55 11 01 28 00 00 00 0C 80 00 CRC
```
- 帧头: AA 55
- 长度: 11 (命令 0x11 + 数据 16 字节 = 17 = 0x11)
- 命令: 11
- 数据: 01(开) 28(亮度40) 00 00 00(RGB) 0C 80(色温3200) 00(模式)
- CRC: 计算值

**接收:**
```
55 AA 91 01 CRC
```
- 帧头: 55 AA
- 响应码: 91
- 状态: 01 (成功)

---

### 6.2 查询体征

**发送:**
```
AA 55 20 CRC
```

**接收:**
```
55 AA A0 01 41 0E 02 46 CRC
```
- 心率: 0x41 = 65 BPM
- 呼吸率: 0x0E = 14 次/分
- 体温: 0x02 0x46 = 582 / 10 = 58.2°C ❌ (示例错误)
