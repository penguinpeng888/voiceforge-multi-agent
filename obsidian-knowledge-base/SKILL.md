# Knowledge to Skill 转化器

## 目的
将知识库中有价值的内容转化为可复用的 AI Skill，实现自我进化。

## 触发条件
- 用户分享有价值的网页/文章
- 发现可复用的工作流或模式
- 需要将外部知识整合到系统中

## 工作流

### 1. 知识提取
从网页/文档中提取：
- 核心概念（是什么）
- 操作步骤（怎么做）
- 关键参数/配置
- 适用场景

### 2. 模式识别
判断是否值得转化为 Skill：
- 是否可复用？（不是一次性信息）
- 是否有标准化流程？
- 是否是高频任务？

### 3. Skill 生成
生成标准格式的 SKILL.md：

```markdown
# Skill 名称

## 描述
一句话说明这个 skill 做什么

## 触发条件
什么情况下使用这个 skill

## 工作流
1. 步骤1
2. 步骤2
3. 步骤3

## 参数/配置
- 参数1: 说明
- 参数2: 说明

## 示例
输入 → 输出

## 注意事项
- 注意点1
- 注意点2
```

### 4. 部署
- 写入 ~/.openclaw/skills/[skill-name]/SKILL.md
- 更新可用技能列表
- 可立即使用

## 使用方法
```
我会分析你分享的内容，提取有价值的知识，并判断是否需要转化为 Skill。
如果值得转化，我会生成标准格式的 SKILL.md 并部署到系统中。
```

## 示例
用户分享一篇 "如何在 Obsidian 中搭建 AI 知识库" 的文章
→ 提取步骤和配置
→ 生成 "obsidian-ai-kb" skill
→ 写入系统后可直接调用
