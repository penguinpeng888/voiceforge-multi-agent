# VoltAgent awesome-design-md

## 核心理念

| 文件 | 用途 |
|------|------|
| **AGENTS.md** | 告诉 AI 如何构建项目 |
| **DESIGN.md** | 告诉 AI 项目应该长什么样 |

## DESIGN.md 结构（9个部分）

1. Visual Theme & Atmosphere - 氛围、密度、设计哲学
2. Color Palette & Roles - 语义颜色 + Hex + 功能角色
3. Typography Rules - 字体家族、完整层级表
4. Component Stylings - 按钮、卡片、输入框、导航（含状态）
5. Layout Principles - 间距、网格、留白
6. Depth & Elevation - 阴影系统、表面层级
7. Do's and Don'ts - 设计规范和反模式
8. Responsive Behavior - 断点、触控、折叠策略
9. Agent Prompt Guide - 快速参考、可用提示词

## 精选网站设计

### AI 工具
- Claude - Anthropic AI，暖色调赤陶土强调色
- Cursor - AI 编程编辑器，深色界面，渐变强调
- ElevenLabs - AI 语音平台，暗色电影感
- Ollama - 本地运行 LLM，终端极简风
- xAI - Elon Musk AI 实验室，黑白未来感

### 开发者工具
- **Linear** - 项目管理，超简洁，紫色强调
- **Vercel** - 前端部署，黑白 precision，Geist 字体
- Raycast - 生产力启动器，深色 Chrome，渐变
- Warp - 现代终端，IDE 风格
- Supabase - 开源 Firebase，深色翡翠主题

### 设计工具
- Figma - 协作设计，多彩活泼
- Framer - 网站构建，黑蓝 bold

## 使用方式

```bash
# 复制 DESIGN.md 到项目
cp design-md/linear/DESIGN.md ./

# 告诉 AI Agent
"build me a page that looks like this"
```
