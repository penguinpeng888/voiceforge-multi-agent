# 妖狼技能清单

> 记录第18章出现的妖狼技能，用于第19章及后续章节

## 基础属性

| 属性 | 值 |
|------|-----|
| 种族 | 妖狼 |
| 境界 | 炼气中期 |
| 体型 | 一人多高，灰黑皮毛 |

## 技能列表

### 1. 狼嚎术（天赋技能）
**效果**：震慑对手心神，让其灵力运转迟滞
**描写**：
```
妖狼忽然仰天长嚎。

"呜——"

声浪滚滚，震得陆沉耳膜发疼。他感觉胸口像是被什么东西重击，呼吸一窒，灵力运转都迟滞了几分。
```

**关键词**：震慑、心神、灵力运转迟滞、声浪滚滚

---

### 2. 獠牙穿刺（天赋技能）
**效果**：将灵力汇聚于獠牙，洞穿护体灵力
**描写**：
```
妖狼落地后没有给陆沉喘息的机会，獠牙上寒光闪烁，再次扑来。

这一次，它张开了血盆大口，獠牙根部泛起淡淡的青色光芒——**獠牙穿刺**！妖狼将灵力汇聚于獠牙之上，锋利度凭空增加了数倍，足以洞穿炼气期修士的护体灵力！
```

**关键词**：獠牙根部、青色光芒、灵力汇聚、洞穿护体

---

### 3. 连击（战斗技巧）
**效果**：快速连续攻击，封住对手退路
**描写**：
```
妖狼的四爪在地上刨动，泥土飞溅。忽然它化作一道灰影，四爪连踏空气中发出"嗖嗖"的破空声。第一爪、第二爪、第三爪——短短三息之间连出七爪，每一爪都角度刁钻封住陆沉的退路。
```

**关键词**：四爪连踏、破空声、角度刁钻、封住退路

---

### 4. 扑杀（致命一击）
**效果**：后腿发力凌空跃起，獠牙利爪同时攻击
**描写**：
```
妖狼后腿发力身体前倾一跃而起——

就是现在！

陆沉没有躲。他盯着妖狼的眼睛，在它跃起的瞬间侧身向旁边一滚。
妖狼扑空撞在树干上发出"砰"的一声闷响。
```

**关键词**：后腿发力、身体前倾一跃而起、侧身翻滚、撞在树干

---

## 技能使用规律

| 顺序 | 技能 | 描述 |
|------|------|------|

| 2 | 狼嚎术 | 震慑心神 |
| 3 | 獠牙穿刺 | 致命攻击 |

## 技能描写模板

### 天赋技能描写
```
[妖兽名]忽然[动作]。

"[叫声]——"

[声浪/光芒效果]，[对对手的影响]。
**技能名**！ [妖兽名]的[天赋技能]，能够[效果]！
```

### 连击描写
```
[妖兽名]的[身体部位]在地上[动作]，[环境效果]。忽然它化作一道[颜色]影，[身体部位]连踏空气中发出"[声音]"的[名词]。第一[攻击]、第二[攻击]、第三[攻击]——短短[时间]之间连出[数字][攻击]，每一[攻击]都[形容词][位置/角度]。[结果]。
```

### 扑杀描写
```
[妖兽名][身体部位]发力[动作]——

就是现在！

[人类角色]没有躲。他盯着[妖兽名]的[部位]，在它[动作]的瞬间[闪避动作]。
[妖兽名]扑空[撞在XXX上]发出"砰"的[声音]。
```

## 使用建议

### 第19章使用
- 遇到新的妖狼时，可以增加 **扑杀** 技能
- 描写陆沉观察到妖狼"身体微微后仰"的细节
- 陆沉利用观察到的规律反击

### 后续章节
- 不同的妖狼可以有不同的技能组合
- 可以加入新的技能（如"血腥撕咬"、"尾扫"等）
- 技能描写要配合战斗节奏
