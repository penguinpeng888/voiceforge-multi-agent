## 常用命令

```bash
# 查看 ontology 数据
python3 scripts/ontology.py list --type Character

# 添加角色
python3 scripts/ontology.py create --type Character --props '{"name":"陆沉","status":"alive"}'

# 查询关系
python3 scripts/ontology.py related --id [ID] --rel 朋友
```
