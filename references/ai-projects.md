# AI 项目参考库

## Claw AI Lab
- **URL**: https://github.com/Claw-AI-Lab/Claw-AI-Lab
- **描述**: AI 原生多智能体研究平台，一个 Dashboard = 一整个研究团队
- **功能**:
  - 通过单个 prompt 创建完整 AI 研究实验室
  - 多智能体并行协作，自动产出：论文 + 代码 + 图表 + 实验日志
  - 三种模式：Explore / Discussion / Reproduce
- **技术栈**: Python + Node.js
- **特点**: 
  - 实时事件流监控
  - 人类在环，可干预、回滚
  - FIFO 调度，最大化计算利用率
- **更新**: 2026-04-07
