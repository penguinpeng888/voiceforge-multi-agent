## 核心架构

### 统一Transformer设计

```
文本Token + 图像Token + 视频Token + 音频Token → 统一处理 → 输出视频+音频
```

**关键特点**：
- 无跨注意力（no cross-attention）
- 无单独音频分支
- 无专用条件网络

### Sandwich 结构

```
[4层模态特定投影] → [32层共享参数] → [4层模态特定投影]
```

- 首尾4层：模态特定投影
- 中间32层：所有模态共享参数

### 多模态融合

- Per-head 学习标量门控（sigmoid 激活）
- 训练稳定性更好

---

## 训练技巧

### DMD-2 蒸馏
- Distribution Matching Distillation v2
- 从 25-50 步 → 8 步
- 无需 Classifier-Free Guidance (CFG)

### MagiCompiler
- 全图编译
- 额外 ~1.2× 加速

### 时间步处理
- 无显式时间步嵌入
- 模型直接从输入 latent 的噪声水平推断去噪状态

## 创新点总结

1. **大一统架构**：文本、图像、视频、音频一个模型
2. **原生音视频**：一次前向传播，同时输出视频和音频
3. **蒸馏优化**：8步生成，质量不降
4. ** Sandwich 设计**：平衡模态特异性和共享知识
