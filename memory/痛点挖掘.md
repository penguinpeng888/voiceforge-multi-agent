# 小说写作软件 - 完整竞品分析

> 更新时间：2026-04-08

---

## 一、目标用户画像

### 核心用户群
| 人群 | 特征 | 痛点 |
|------|------|------|
| **网文作者** | 日更3000-10000字 | 卡文、断更、剧情崩盘 |
| **新人作者** | 想写但不会写 | 没大纲、没灵感、字数不够 |
| **兼职作者** | 晚上/周末写 | 时间碎片化、需要同步 |
| **小说爱好者** | 写着玩 | 工具太复杂、学不会 |

### 用户规模
- 中国网文作者：**1700万+**（2025年数据）
- 活跃作者：约 200-300 万
- 付费意愿：中等（愿意为效率付费）

---

## 二、竞品分析

### 竞品 1：笔灵AI写作
| 维度 | 分析 |
|------|------|
| **定位** | AI一键生成小说 |
| **优点** | 网文套路最深、情节生成快、大纲到正文全流程 |
| **缺点** | AI味重、缺乏深度创作体验、收费贵 |
| **价格** | 付费会员制 |
| **用户痛点** | 生成内容套路化、不能个性化定制 |

### 竞品 2：橙瓜码字
| 维度 | 分析 |
|------|------|
| **定位** | 网文写作工具 |
| **优点** | 同步更新、多平台支持、码字统计 |
| **缺点** | 功能太杂、广告多、界面老旧 |
| **价格** | 免费+会员 |
| **用户痛点** | 广告烦人、功能太多不会用 |

### 竞品 3：Scrivener
| 维度 | 分析 |
|------|------|
| **定位** | 专业写作软件 |
| **优点** | 功能强大、软木塞模式、导出格式多 |
| **缺点** | 中国区售价358元太贵、学习曲线陡峭、全英文 |
| **价格** | 358元（一次性） |
| **用户痛点** | 贵！且不会用 |

### 竞品 4：Ulysses
| 维度 | 分析 |
|------|------|
| **定位** | Markdown写作 |
| **优点** | 极简优雅、聚焦写作、主题漂亮 |
| **缺点** | 仅Mac/iOS、贵、无中文优化 |
| **价格** | 283元/年 |
| **用户痛点** | Windows不能用、国内用户少 |

### 竞品 5：纯纯写作
| 维度 | 分析 |
|------|------|
| **定位** | 极简写作APP |
| **优点** | 流畅、极简、即时保存 |
| **缺点** | 功能太少、无大纲/设定管理 |
| **价格** | 免费+捐赠 |
| **用户痛点** | 只能写作，不能管理大纲/世界观 |

### 竞品 6：Writeathon
| 维度 | 分析 |
|------|------|
| **定位** | 写作模式+计时 |
| **优点** | 写作目标、专注模式 |
| **缺点** | 功能单一、无AI辅助 |
| **价格** | 免费+会员 |
| **用户痛点** | 缺乏AI和大纲管理 |

## 三、竞品痛点总结

| 痛点 | 影响人群 | 现有解决方案 |
|------|----------|--------------|
| **收费太贵** | 兼职/新人 | 无好替代 |
| **功能太复杂** | 新人 | 无好替代 |
| **广告太多** | 所有人 | 无好替代 |
| **无AI辅助** | 卡文作者 | 笔灵等AI工具 |
| **不能管理世界观** | 资深作者 | Scrivener（贵） |
| **仅Mac可用** | Windows用户 | 无 |
| **全英文** | 中国用户 | 无 |
| **界面老旧** | 年轻作者 | 无 |
