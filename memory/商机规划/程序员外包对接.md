## 二、接单话术

### 客户问候
```
您好！请问有什么开发需求？

为了给您准确报价，请告诉我：
1. 您想做什么类型的项目？（网站/小程序/APP）
2. 主要功能有哪些？（比如：展示产品、在线购买、用户登录等）
3. 有没有参考对象？
4. 您的预算大概是多少？
```

### 需求评估
```
好的，根据您的需求，我给您分析一下：

功能：XXX
工期：约X天

1. 先付30%定金：XXX元
2. 初版完成后付30%：XXX元
3. 确认无误付尾款：XXX元

您看可以吗？
```

## 三、寻找程序员

### 闲鱼搜索关键词
```
开发者 个人开发者 网站开发 小程序开发
程序员 接单 兼职 承接开发
技术外包 网站制作 APP开发
```

### 筛选标准
| 标准 | 及格线 | 优秀 |
|------|--------|------|
| 好评率 | 95%+ | 98%+ |
| 案例 | 有类似 | 多个相关 |
| 响应速度 | 2小时内 | 30分钟内 |
| 沟通 | 细问需求 | 主动建议 |

### 测试问题（辨别是否靠谱）
```
1. "这个功能怎么做？"（看回答是否专业）
2. "大概需要多久？"（看是否有经验）
3. "如果做不出来怎么办？"（看是否负责）
4. "能先做个demo吗？"（看是否愿意测试）
```

### 付款阶段
```
第1阶段（确认需求）：不付款
第2阶段（开始开发）：收30%
第3阶段（初版完成）：收30%
第4阶段（确认尾款）：收40%
```

### 避坑清单
❌ 不要找报价低于市场价50%的
❌ 不要信"保证做好，包满意"
❌ 不要一次性付全款
❌ 不要找好评都是"很棒""很好"的（刷的）
❌ 不要找从来不问需求就接单的
