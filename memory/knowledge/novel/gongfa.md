# 功法设定

## 主角功法
| 功法 | 类型 | 功效 |
|------|------|------|
| 噬灵诀 | 禁术 | 吞噬一切力量，燃烧潜力换战力，事后虚弱3小时 |
| 经脉淬炼术 | 锻体 | 锤炼经脉，可炼化妖丹 |
| 流云十三式 | 剑法 | 13式，攻防一体 |

## 流云十三式
1. 流云出岫 - 身法轻灵
2. 云起龙骧 - 攻击力强
3. 云过无痕 - 身法飘忽
4. 云卷云舒 - 转身灵活
5. 流云随风 - 借力打力
6. 云破天开 - 破防攻击
7-13. 待解锁

## 噬灵诀
- 爷爷自创《噬灵》演变而来
- 可炼化毒素转化为灵气
- 长期修炼可强化经脉和丹田
