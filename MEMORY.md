### 《功法体系》设定

> 后续创作必须保持功法名称、功效、使用条件一致！

#### 一、主角功法

| 功法 | 类型 | 功效 | 章节 | 使用条件 |
|------|------|------|------|----------|
| **经脉淬炼术** | 锻体功法 | 锤炼经脉，提升战斗力；炼化妖丹增强经脉 | 第18章 | 无限制，可日常修炼 |
| **流云十三式** | 剑法 | 13式剑法，攻防一体 | 第18章 | 需短剑或剑形法器 |
| **噬灵诀** | 禁术 | 燃烧潜力，短暂提升战力 | 第19章 | 消耗精血，事后虚弱 |
| **燃脉法** | 禁术 | 配合噬灵诀使用 | 第19章 | 需与噬灵诀同用 |

#### 二、功法详解

**《经脉淬炼术》**
- 类型：锻体功法
- 功效：锤炼经脉，提升战斗力；可将妖丹炼化为己用
- 修炼方式：日常运转 + 妖丹炼化
- 副作用：炼化妖丹时剧痛无比

**《流云十三式》**
- 类型：剑法武技
- 功效：身法与攻击结合，共13式
- 修炼方式：需要不断实战磨练
- 限制：需要剑形武器

**《噬灵诀》**
- 类型：禁术
- 功效：燃烧潜力换取短暂战力暴涨
- 使用代价：消耗精血，事后虚弱3小时
- 使用场景：生死关头

**《燃脉法》**
- 类型：禁术辅助
- 功效：配合噬灵诀，爆发更强力量
- 使用代价：经脉受损，需修养
- 必须与噬灵诀同时使用

#### 三、术法与功法关系图

```
经脉淬炼术（基础）
    ↓ 锤炼经脉
流云十三式（战斗）
    ↓ 实战检验
噬灵诀 + 燃脉法（底牌）
    ↓ 生死关头使用
```

#### 四、后续创作注意事项

1. **功法名称必须一致**：
   - 经脉淬炼术 ≠ 经脉淬炼功法
   - 噬灵诀 ≠ 噬灵大法
   - 燃脉法 ≠ 燃脉术

2. **功效必须统一**：
   - 经脉淬炼术不能突然增加"吸收灵气"功能
   - 噬灵诀的虚弱时间固定为3小时

3. **招式名称必须一致**：
   - 详见《流云十三式设定表》

4. **新功法添加规则**：
   - 新功法需在设定文档中提前说明
   - 不能与已有功法功效冲突
   - 需说明修炼条件和副作用

## 四、第十八章总结（磨砺·一）

**标题**: 磨砺（一）
**字数**: ~8000字
**核心情节**:
- **开篇**: 陆沉突破炼气五层三天，但缺乏实战经验，想起那晚差点被筑基期杀死的恐惧
- **决心**: 决定外出猎杀妖兽，靠自己变强
- **初战**: 遭遇炼气中期妖狼，第一次真正面对妖兽
  - 恐惧→冷静→观察→反击
  - 观察到妖狼扑击前身体后仰的规律
  - 利用规律反杀，获得第一颗妖丹
- **炼化**: 第一次炼化妖丹，妖力狂暴，经脉剧痛
  - 咬牙坚持，险些放弃但想起周师兄咬牙撑过
  - 经脉比之前坚韧一丝
- **结尾**: 夜幕降临，陆沉对着沉睡的猪说"我会变强的"

**应用基因**:
- 战斗描写（起承转合：试探→劣势→观察→反杀）
- 心理活动（恐惧→冷静→感悟"怕死的人最容易死"）
- 感官细节（血腥味、疼痛感、劫后余生的虚脱感）

**修改要点**:
1. "妖兽能猎杀妖兽，人为什么不能？" → "人为何不能？"（避免重复）
2. 加"怕死的人，才最容易死。"（强化心理转折）
3. 加"想放弃→咬牙坚持"的炼化瞬间
4. 加"等从炼化的余韵中回过神来，洞府外的天光已经暗了。"

  - 妖丹积累到二十多颗

  - 找到七寸处浅色鳞片

- 战斗描写（多种妖兽，各有特点和战斗方式）
- 感官细节（妖蛇的竖瞳、妖鹰俯冲的风声、妖蟒的力量）

**修改要点**:
1. 妖蛇战扩充300字：增加被追时的恐惧、妖蛇戏弄

4. 时间过渡：增加"休整两日后""伤好之后""又过了几天"等过渡句
5. 妖蟒战扩充200字：增加被甩动的失重感、剑被打滑、愣几息
