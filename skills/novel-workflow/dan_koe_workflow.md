# Dan Koe 内容挖掘工作流

> 视频下载 → 转录 → 翻译 → 中文改写 → 发布

---

## 📋 工作流程

```
1️⃣ 下载视频       ./scripts/youtube-batch.sh <links.txt>
2️⃣ 转录英文       whisper <video>.mp3 --model base
3️⃣ 翻译中文       调用 LLM 翻译
4️⃣ 中文改写       调用 LLM 改写成适合中国人风格
5️⃣ 发布文案       生成最终中文版本
```

---

## 🎯 目标输出

| 输出文件 | 说明 |
|---------|------|
| `dan-koe-原始翻译.txt` | 直译版本 |
| `dan-koe-改写文案.txt` | 符合中国人阅读习惯 |
| `dan-koe-发布词.txt` | 可直接发布的版本 |

---

## 📝 中文改写原则

| 维度 | 要求 |
|------|------|
| **语气** | 口语化、接地气 |
| **案例** | 换成中国本土案例 |
| **表达** | 符合中文阅读习惯 |
| **长度** | 可适当精简或扩展 |

---

## 🚀 启动方式

### 方式1: 等 cron（周三10:00）
自动执行，无需干预

### 方式2: 手动立即执行
提供视频链接，立即处理
