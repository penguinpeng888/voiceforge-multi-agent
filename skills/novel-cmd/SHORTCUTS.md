# 快捷命令工具

这个工具用于快速执行小说创作相关任务。

## 命令格式

在对话中输入以下格式，我会直接执行：

```
!审核 <章节号>
!检查 <章节号>
!发送 <章节号>
!状态
```

## 使用示例

### 审核章节
```
!审核 16
```
执行：发送第16章给审核代理，返回评分和整改建议。

### 运行逻辑检查
```
!检查 16
```
执行：运行 logic-checker.py 脚本检查第16章。

### 发送章节
```
!发送 16
```
执行：通过 Telegram 发送第16章文件给用户。

### 查看状态
```
!状态
```
执行：显示当前系统状态和待办事项。

## 工作流

```
写章节完成
    ↓
!检查 16    # 本地逻辑检查
    ↓
!审核 16    # 代理审核
    ↓
查看评分/修改
    ↓
循环直到9分以上
    ↓
!发送 16    # 定稿发送
```

## 前置条件

- 审核代理：`agent:novel-editor:main`（已创建）
- 章节目录：`/root/.openclaw/workspace/剑起青云/`
- 逻辑检查脚本：`skills/novel-logic-checker/scripts/logic-checker.py`

## 注意事项

- `!审核` 命令会调用审核代理，需要几秒钟生成审核报告
- 如果审核分数 < 9，需要修改后重新审核
- 建议先用 `!检查` 做本地检查，再 `!审核` 做代理审核
