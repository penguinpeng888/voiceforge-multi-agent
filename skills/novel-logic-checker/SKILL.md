# Novel Logic Checker

小说逻辑检查工具 - 四大维度全面审核

## 功能

1. **内部逻辑检查** - 故事世界的自洽性
2. **外部逻辑检查** - 与现实世界的关联
3. **情节人物逻辑** - 故事推进的动力
4. **叙事结构逻辑** - 讲述故事的方式
5. **修真小说特有检查** - 境界/功法/战斗/势力
6. **评分机制** - 9分以上可定稿

## 使用方法

```bash
# 逻辑检查
python3 scripts/logic-checker.py --file /path/to/chapter.md --chapter 16

# 详细报告
python3 scripts/logic-checker.py --file /path/to/chapter.md --chapter 16 --detail

# 仅修真检查
python3 scripts/logic-checker.py --file /path/to/chapter.md --xianxia-only
```

## 四大核心维度

### 一、内部逻辑：故事世界的自洽性
- 规则的一致性：功法耗蓝/灵气设定统一
- 设定的限制：强大能力有相应限制
- 社会与文化的合理性：修真体系自圆其说

### 二、外部逻辑：与现实世界的关联
- 自然科学法则：受伤恢复符合常理
- 人类行为与心理：情感转变有铺垫
- 历史与社会常识：货币/资源体系合理

### 三、情节与人物逻辑
- 人物动机与成长：有足够动机支撑
- 情节因果链：发展遵循因果关系
- 信息对称性：决策基于已知信息

### 四、叙事与结构逻辑
- 时间线连贯：无时间矛盾
- 视角一致性：描写符合视角规则
- 伏笔与呼应：开篇结尾有交代

## 修真小说特有检查

| 检查项 | 说明 |
|--------|------|
| 境界体系 | 炼气→筑基→金丹→元婴→化神→大乘→渡劫 |
| 功法体系 | 噬灵诀特点、杂灵根限制 |
| 战斗逻辑 | 等级压制、战斗消耗、以弱胜强 |
| 势力逻辑 | 血影殿 vs 御剑宗 vs 通宝商会 |

## 评分标准

| 分数 | 等级 | 结论 |
|------|------|------|
| 9.0+ | 优秀 | 可定稿 |
| 8.0-8.9 | 良好 | 稍作修改 |
| 7.0-7.9 | 一般 | 需要修改 |
| 6.0-6.9 | 及格 | 需要大改 |
| 6.0- | 不及格 | 需重写 |

## 实用技巧

1. 绘制时间线与关系图
2. 角色访谈（问角色"为什么这么做"）
3. 请他人阅读（旁观者更容易发现漏洞）
4. 拉长修改间隔（写完放一放再审）

## 核心原则

> 逻辑为故事服务
>
> 不要因追求极致的逻辑而牺牲故事的戏剧性和情感力量。
