# 写作技法创作清单

## 战斗场景检查清单

写作前：
- [ ] 这场战斗的目的是什么？（推动剧情/展现成长/解决矛盾）
- [ ] 双方实力对比如何？（等级、资源、底牌）
- [ ] 主角有什么可用资源？（符箓、丹药、法宝、阵法）
- [ ] 有什么环境因素可以利用？
- [ ] 战斗前有没有铺垫？（伏笔、暗示）
- [ ] 有没有旁观者需要"扩音"？

写作中：
- [ ] 感官描写到位了吗？（声音、光线、震荡）
- [ ] 镜头切换了吗？（远景→中景→特写）
- [ ] 节奏感如何？（发牌理论：小→中→大→王炸）
- [ ] 有没有见招拆招的动作博弈？
- [ ] 主角有没有展现出智取而非硬拼？

写作后：
- [ ] 有没有"世界余波"？（对主角/他人/剧情的影响）
- [ ] 战斗结果是否符合预期？（策略/意志/底牌）
- [ ] 有没有为下一场战斗埋下伏笔？

---

## 心理场景检查清单

写作前：
- [ ] 这个角色的核心执念是什么？
- [ ] 他/她当前面临什么困境？
- [ ] 有没有内心冲突？（理性vs感性、理智vs情感）
- [ ] 角色之间的关系如何？（利益？情感？）

写作中：
- [ ] 内心独白是否真实？（不是"演戏"，而是真实想法）
- [ ] 有没有多层心理活动？（表面一层，内心多层）
- [ ] 是否避免了直白说教？（用心象投影代替"他很难过"）
- [ ] 反派的动机是否可理解？

写作后：
- [ ] 这个心理转折是否影响角色成长？
- [ ] 是否与整体故事主题呼应？

---

## 环境描写检查清单

写作前：
- [ ] 这个场景的功能是什么？（交代背景/铺垫剧情/营造氛围）
- [ ] 需要什么情绪基调？（悬疑/温馨/悲壮）
- [ ] 环境能否参与剧情？（枷锁/武器/暗示）

写作中：
- [ ] 功能性：是否一句到位传达信息？
- [ ] 氛围感：是否做到定调统一？
- [ ] 参与度：环境有没有成为角色/战斗的一部分？
- [ ] 是否避免了"蓝天白云绿草地"的枯燥描写？

写作后：
- [ ] 是否达到金字塔的某一层？（白描/渲染/隐喻）
- [ ] 环境与角色心境是否呼应？

## 《剑起青云》特有问题清单

### 陆沉相关
- [ ] 杂灵根的设定是否影响了战斗方式？
- [ ] 灭门之仇的执念是否贯穿心理描写？
- [ ] 三年杂役生涯是否体现在言行细节中？
- [ ] 噬灵诀的独特性是否展现出来？

### 玄冥相关
- [ ] 猪形态时是否埋下伏笔？
- [ ] 妖王身份揭晓时是否有震撼感？
- [ ] 与陆沉的羁绊是否有层次感？

### 战斗相关
- [ ] 等级压制是否严谨？
- [ ] 越级挑战是否靠智取而非金手指？
- [ ] 周家敌人的险恶是否体现出来？

---

## 写作后自查

- [ ] 战斗场面是否"播片"？（读者能脑补画面）
- [ ] 心理描写是否"走心"？（不是流水账）
- [ ] 环境描写是否"入境"？（身临其境）
- [ ] 三者是否有机结合？（而非割裂）
