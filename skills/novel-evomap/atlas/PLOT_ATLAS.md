# Novel Atlas - 剧情图谱系统

## 概述

剧情图谱是 EvoMap 的核心组件之一，用于管理小说的：
- **剧情线** - 主线与支线的发展
- **人物关系** - 角色间的复杂关系
- **伏笔追踪** - 埋下的伏笔与回收状态
- **时间线** - 事件发生的时序

与基因库的区别：
- **基因库** - 存储"怎么写"的技法
- **剧情图谱** - 存储"写什么"的内容

## 脚本功能

### atlas_manager.py

```python
from atlas_manager import AtlasManager

atlas = AtlasManager()

# 查看剧情线
plot = atlas.get_plot()
print(json.dumps(plot, ensure_ascii=False, indent=2))

# 查看人物关系
characters = atlas.get_characters()
print(json.dumps(characters, ensure_ascii=False, indent=2))

# 获取伏笔列表
foresights = atlas.get_foresights()
for f in foresights:
    print(f"{f['伏笔']}: {f['当前状态']}")

# 查看时间线
timeline = atlas.get_timeline()
for t in timeline:
    print(f"{t['时间点']}: {t['事件']}")
```

### 交互命令

```bash
# 查看剧情线
python scripts/atlas_manager.py plot

# 查看人物关系
python scripts/atlas_manager.py characters

# 查看伏笔
python scripts/atlas_manager.py foresights

# 查看时间线
python scripts/atlas_manager.py timeline

# 查看完整图谱
python scripts/atlas_manager.py all

# 添加伏笔
python scripts/atlas_manager.py add-foresight "新伏笔" "第X章" "描述"

# 更新状态
python scripts/atlas_manager.py update-foresight "伏笔ID" "已完成"
```
