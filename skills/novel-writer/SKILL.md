# Novel Writer - 爽文写作助手

基于热门爽文规律总结的创作工具，支持修真、穿越、搞笑、都市、历史、武侠、官场、校园、赌博等多种类型。

## 核心功能

1. **爽点分析** - 根据类型提取核心爽点公式
2. **节奏把控** - 打脸、升级、装X的时机与频率
3. **金句生成** - 符合类型风格的经典台词
4. **大纲生成** - 从粗纲到细纲的完整规划
5. **人物模板** - 各类型热门角色设定

## 支持的小说类型

| 类型 | 核心爽点 | 代表场景 | 节奏特点 |
|------|----------|----------|----------|
| **修真** | 境界碾压、奇遇不断 | 修炼、斗法、夺宝 | 升级+战斗循环 |
| **穿越** | 现代知识降维打击 | 官场、商场、战场 | 知识变现 |
| **搞笑** | 反转、吐槽、无厘头 | 日常、对话、冲突 | 高频笑点 |
| **都市** | 财富、权力、女人 | 暧昧、争斗、逆袭 | 现实映射 |
| **历史** | 改变历史、收服名将 | 争霸、改革、战争 | 宏大叙事 |
| **武侠** | 江湖恩怨、武功秘籍 | 复仇、历练、奇遇 | 恩怨情仇 |
| **官场** | 权谋、晋升、打脸 | 勾心斗角、派系争斗 | 步步为营 |
| **校园** | 逆袭、暧昧、炫富 | 考试、追女、校霸 | 青春热血 |
| **赌博** | 逆天改命、神操作 | 赌局、猜牌、逆转 | 紧张刺激 |

## 使用方法

直接告诉我你想要的内容，例如：

- "给我一个修真小说的200章大纲"
- "生成一个都市打脸情节模板"
- "写一段金丹期碾压筑基期的战斗场面"
- "帮我设计一个贪财好色的反派角色"

## 爽文核心公式

### 通用爽点模板

```
被打脸 → 隐藏实力暴露 → 对方震惊 → 彻底碾压 → 收获好处
```

### 节奏黄金法则

| 节奏点 | 频率 | 字数占比 |
|--------|------|----------|
| 小打脸 | 每10章 | 5% |
| 中打脸 | 每30章 | 10% |
| 大打脸 | 每100章 | 20% |
| 奇遇 | 每20章 | 8% |
| 暧昧 | 每15章 | 5% |

### 角色成长模板

```
废材/底层 → 获得金手指 → 小试牛刀 → 被打压 → 爆发打脸 → 升级 → 更大的舞台
```

## 输出格式

我会根据你的需求输出：
- 章节大纲（粗纲/细纲）
- 场景描写（战斗/暧昧/打脸等）
- 人物设定卡
- 金句/台词
- 爽点分析
